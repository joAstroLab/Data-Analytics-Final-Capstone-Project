# Final Capstone Project - Car Resale Value Analysis 
This project aims to investigate factors influencing car resale value, focusing on market trends, usage history, depreciation, mileage-price relationships, market demand and fuel type market share. The dataset used is from Kaggle: vehicle-dataset-from-cardekho. The project follows a typical data analysis workflow, including data loading, initial overview, data cleaning and pre-processing (handling missing values, duplicates and data type conversions), exploratory data analysis (EDA) with filtering and aggregation, and various visualizations (univariate, bivariate and multivariate analysis) to generate insights and a final summary. Key findings include the influence of brand, transmission, fuel type, vehicle age, engine capacity, seating capacity, ownership history and mileage on resale value.
